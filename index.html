Masters Thesis

Title: "Design and Development of Thin Film Sensors"

Thin film sensor's ( thermocouple and strain gauge ) deposition was carried out by using DC & RF magnetron sputtering and the parameters for deposition were optimized on Kaptan tape. Insulation and Insulation and bond coat were studied for INCONEL (gas turbine blade material) and the deposition parameters for these on INCONEL were optimized. Characterization was done by SEM and X-Ray diffraction techniques to ensure proper layers of bond and insulation coats.Strain gauge and thermocouple are then deposited on the insulation coat and the parameters were optimized.

Bachelors Thesis

Title: Estimation of Structural Parameters of Ge-C alloys using cluster variation method

Experimental determination of phase diagrams is a time consuming and costly task. This becomes even more pronounced as the number of components increases. Cluster variation method has been employed in this project to determine the free energies of a 2D binary alloy system (Ge-C). Using these determined free energies, the phase diagram has been plotted using Mathematica. This calculated phase diagram has been compared with the phase diagram obtained using ThermoCalc software.
